# Introduction to ClickLens

ClickLens is a modern, powerful, and user-friendly web interface for managing and monitoring [ClickHouse](https://clickhouse.com/) databases. It provides a comprehensive suite of tools for developers and administrators to interact with their ClickHouse clusters efficiently.

![ClickLens Dashboard](/dashboard-preview.png)

## Why ClickLens?

ClickHouse is an incredible analytical database, but managing it via CLI or basic tools can be challenging. ClickLens bridges the gap by offering:

- **Visual SQL Console**: Write, execute, and analyze queries with ease.
- **Real-time Monitoring**: Keep an eye on your cluster's health, query performance, and resource usage.
- **Schema Explorer**: Navigate through databases, tables, and system tables.
- **Access Control**: Manage users and roles directly from the UI.
- **Modern Tech Stack**: Built with Next.js, Shadcn UI, and Tailwind CSS for a fast and responsive experience.

## Key Goals

- **Simplicity**: Make complex ClickHouse operations accessible.
- **Performance**: Fast UI for a fast database.
- **Observability**: Provide deep insights into what's happening inside the cluster.
